# 💪 增肌训练计划 | Workout Plan

一个响应式的增肌训练计划网页，包含训练安排、饮食建议和动作详解。

A responsive muscle-building workout plan page with training schedule, nutrition guidance, and exercise details.

## 🌐 在线预览 | Live Demo

**访问网站 | Visit Site**: [https://folozz.github.io/Workout-Plan/](https://folozz.github.io/Workout-Plan/)

## ✨ 主要特性 | Key Features

- 🎨 玻璃态 UI + 渐变视觉 + 动画过渡
- 📱 完全响应式（桌面 / 平板 / 手机）
- 🏋️ 4 天分化训练计划（上肢推、上肢拉、肩臂、下肢）
- 🍽️ 每日营养目标与饮食时间建议
- 🎬 动作弹窗使用真人动作分解 GIF 演示
- ⏭️ 弹窗底部固定「上一个 / 下一个」按钮，无需下滑即可切换动作

## 📋 训练安排 | Training Split

- **周一**: 胸部 + 三头肌（推力）
- **周二**: 背部 + 二头肌（拉力）
- **周四**: 肩部 + 手臂强化
- **周五**: 下肢训练

## 🛠️ 技术栈 | Tech Stack

- HTML5
- CSS3（Grid / Flexbox / 动画 / 变量）
- Vanilla JavaScript
- Google Fonts（Plus Jakarta Sans）

## 🚀 本地运行 | Local Run

1. 克隆仓库：

```bash
git clone https://github.com/Folozz/Workout-Plan.git
```

2. 进入目录：

```bash
cd Workout-Plan
```

3. 直接用浏览器打开 `index.html`，或使用本地静态服务器运行。

## 🧩 自定义说明 | Customization

- 训练动作数据：编辑 `index.html` 中的 `exerciseData`
- 每日动作顺序：编辑 `index.html` 中的 `dayExercises`
- 动作示例图：替换 `exerciseData` 里的 `image` 链接（支持本地路径或外链）
- 主题样式：修改 `:root` 下的 CSS 变量
