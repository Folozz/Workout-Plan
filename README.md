# 💪 增肌训练计划 | Workout Plan

一个精美的响应式健身训练计划网页，专为增肌目标设计。

A beautiful responsive workout plan webpage designed for muscle building goals.

## 🌐 在线预览 | Live Demo

**访问网站 | Visit Site**: [https://folozz.github.io/Workout-Plan/](https://folozz.github.io/Workout-Plan/)

## ✨ 特性 | Features

- 🎨 **现代化设计** - 采用玻璃态设计风格，渐变色彩，流畅动画
- 📱 **完全响应式** - 完美适配桌面、平板和移动设备
- 🏋️ **4天训练计划** - 科学的上下肢分化训练方案
- 🍽️ **营养指导** - 详细的每日饮食计划和宏量营养素目标
- 💡 **动作详解** - 点击动作查看详细说明和要点
- ⚡ **交互体验** - 平滑滚动、悬停效果、模态框展示

## 📋 训练计划概览 | Training Overview

### 训练安排 | Training Schedule
- **周一**: 胸部 + 三头肌
- **周二**: 背部 + 二头肌
- **周四**: 肩部 + 手臂强化
- **周五**: 腿部训练

### 营养目标 | Nutrition Goals
- **热量**: 2600 kcal/天
- **蛋白质**: 130g
- **碳水化合物**: 380g
- **健康脂肪**: 75g

## 🛠️ 技术栈 | Tech Stack

- HTML5
- CSS3 (现代CSS特性：Grid、Flexbox、动画、渐变)
- Vanilla JavaScript
- Google Fonts (Plus Jakarta Sans)

## 🚀 本地运行 | Local Development

1. 克隆仓库 | Clone the repository:
```bash
git clone https://github.com/Folozz/Workout-Plan.git
```

2. 进入目录 | Navigate to directory:
```bash
cd Workout-Plan
```

3. 打开文件 | Open the file:
- 直接在浏览器中打开 `index.html`
- 或使用本地服务器（推荐）
